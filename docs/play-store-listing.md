# GymLog — Play Store Listing

## App title

GymLog: Registro de Entrenamientos

## Short description (≤80 chars)

Registra tus entrenamientos de fuerza con analíticas avanzadas y offline-first.

## Long description (≤4000 chars)

**GymLog** es la app de entrenamiento para quienes toman en serio la hipertrofia y la fuerza. Diseñada con arquitectura **offline-first**: funciona sin conexión y sincroniza cuando hay red.

### ¿Qué puedes hacer?

🏋️ **Registro de entrenamientos**

- Registra ejercicios, series, repeticiones y pesos
- Biblioteca completa de ejercicios con búsqueda
- Plantillas personalizadas para tus rutinas habituales
- Notas por sesión y por ejercicio

📊 **Analíticas avanzadas**

- Gráficos de progreso por ejercicio
- Volumen semanal por grupo muscular (radar chart)
- Cálculo de 1RM con fórmula Brzycki
- Seguimiento de fatiga neuromuscular

⏱️ **Temporizador de descanso**

- Temporizador configurable entre series
- Notificaciones incluso con la pantalla apagada
- Vibración y alarma sonora personalizables

🔒 **Seguridad**

- Acceso con huella dactilar o Face ID (biometría nativa)
- Tus datos viajan siempre cifrados (HTTPS)
- Sincronización segura con Supabase

🌍 **Multiidioma**

- Disponible en Español e Inglés
- Detección automática del idioma del dispositivo

⚡ **Rendimiento**

- Carga instantánea gracias al modo offline-first
- Sin anuncios, sin compras dentro de la app
- Interfaz edge-to-edge optimizada para Android moderno

## English version

### Short description

Track your strength workouts with advanced analytics. Offline-first gym log.

### Long description

**GymLog** is the workout tracker built for hypertrophy and strength athletes. **Offline-first** — works without internet, syncs when connected.

### Features

🏋️ **Workout Logging**

- Log exercises, sets, reps and weights
- Full exercise library with search
- Custom routine templates
- Per-session and per-exercise notes

📊 **Advanced Analytics**

- Progress charts per exercise
- Weekly volume by muscle group (radar chart)
- 1RM calculator (Brzycki formula)
- Neuromuscular fatigue tracking

⏱️ **Rest Timer**

- Configurable rest timer between sets
- Notifications even with screen off
- Customizable vibration and sound

🔒 **Security**

- Fingerprint / Face ID login (native biometrics)
- All data transmitted over HTTPS
- Secure sync via Supabase

🌍 **Multilanguage**

- Spanish and English
- Auto-detects device language

⚡ **Performance**

- Instant load via offline-first architecture
- No ads, no in-app purchases
- Edge-to-edge UI for modern Android
